# 🏎️ 2D Endless Car Racing Game

A fully responsive, browser-based 2D racing game built entirely from scratch with **HTML, CSS, and Vanilla JavaScript**. Dodge enemy vehicles, race for high scores, and enjoy the smooth top-down driving experience!

## 🎮 Features
- **Endless Gameplay:** The longer you survive, the higher your score climbs. Enemies spawn faster and the speed increases dynamically.
- **Turbo Race Mode:** Hold the `RACE🏁` button to double your speed and earn points twice as fast!
- **Sleek CSS Graphics:** No canvas or external images are used. The neon cars, glowing taillights, and immersive moving environment are all meticulously crafted using raw CSS gradients and shadows.
- **Cross-Platform Responsive Design:** Fully playable on Desktop (using Keyboard) and Mobile setups (using touch-controls). 
- **Accurate Hitboxes:** Built with native `getBoundingClientRect` for precise and fair crash collision detection.
- **Clean Architecture:** Professionally organized code splitting HTML structure, CSS aesthetics, and JavaScript game loops.

## 🚀 How to Play
You can play immediately with zero installation required.
1. Download or clone this repository.
2. Open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari).
3. **Desktop Controls:** 
   - `Left Arrow` / `A` = Steer Left
   - `Right Arrow` / `D` = Steer Right
   - `Up Arrow` / `W` / `Space` = RACE (Turbo Speed)
4. **Mobile Controls:** Tap the large interactive buttons underneath the game screen.

## 🛠️ Built With
- **HTML5** (Structure)
- **CSS3** (Styling, Animations, Glassmorphism UI)
- **Vanilla JavaScript** (Game Engine Loop `requestAnimationFrame`, Logic, Collision Math)
- *Absolute Zero Dependencies, No Game Engines or Frameworks.*
